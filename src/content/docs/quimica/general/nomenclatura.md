# Nomenclatura Química Inorgánica

La nomenclatura tiene como objetivo que el nombre de una sustancia nos indique su fórmula y viceversa. Los tres sistemas más utilizados son:

## 1. Nomenclatura Sistemática (IUPAC / Estequiométrica)
Se basa en el uso de prefijos griegos para indicar el número de átomos de cada elemento presente en la molécula.

* **1:** mono- (frecuentemente omitido)
* **2:** di-
* **3:** tri-
* **4:** tetra-
* **5:** penta-

**Ejemplo:** $Fe_2O_3 \longrightarrow$ **Tri**óxido de **di**hierro.

---

## 2. Nomenclatura de Stock
Se nombra el compuesto y, al final, se indica el **número de oxidación** del elemento central en números romanos y entre paréntesis. Si el elemento solo tiene un estado de oxidación, este no se escribe.

**Ejemplo:** $Fe_2O_3 \longrightarrow$ Óxido de hierro **(III)**.



---

## 3. Nomenclatura Tradicional (Funcional)
Utiliza prefijos y sufijos para indicar los estados de oxidación. Aunque la IUPAC la considera obsoleta para muchos compuestos, se sigue usando mucho en ácidos y sales comunes.

| Estados de Oxidación | Prefijo / Sufijo | Ejemplo ($Cl$) |
| :--- | :--- | :--- |
| El menor (de 4) | **hipo - oso** | Ácido **hipo**clor**oso** ($+1$) |
| Menor (de 2 o 3) | **- oso** | Ácido clor**oso** ($+3$) |
| Mayor (de 2 o 3) | **- ico** | Ácido clór**ico** ($+5$) |
| El mayor (de 4) | **per - ico** | Ácido **per**clór**ico** ($+7$) |

---

## Reglas por Función Química

### A. Óxidos (Elemento + Oxígeno)
El oxígeno siempre actúa con número de oxidación **$-2$**.
* $PbO_2$:
    * **Sistemática:** Dióxido de plomo.
    * **Stock:** Óxido de plomo (IV).
    * **Tradicional:** Óxido plúmbico.

### B. Hidruros (Metal + Hidrógeno)
El hidrógeno actúa con **$-1$**.
* $CaH_2$:
    * **Sistemática:** Dihidruro de calcio.
    * **Stock:** Hidruro de calcio.

### C. Hidrácidos (No Metal + Hidrógeno)
En solución acuosa se usa el prefijo **Ácido** y el sufijo **-hídrico**.
* $HCl_{(ac)}$: Ácido clorhídrico.
* $H_2S_{(ac)}$: Ácido sulfhídrico.



---

## Resumen Comparativo: El caso del Hierro ($Fe$)
El hierro tiene estados $+2$ y $+3$. Veamos cómo cambia su nombre según el sistema para el óxido con estado $+3$:

$$ Fe_2^{+3}O_3^{-2} $$

1.  **Sistemática:** Trióxido de dihierro.
2.  **Stock:** Óxido de hierro (III).
3.  **Tradicional:** Óxido férrico (sufijo *-ico* por ser el mayor).
