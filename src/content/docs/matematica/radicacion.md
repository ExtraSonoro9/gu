title: Radicación


---


La radicación es la operación que consiste en buscar un número que multiplicado por sí mismo tantas veces como indica el índice, de como resultado el radicando.

## Definición y Elementos
$$\sqrt[n]{a} = b \iff b^n = a$$

* **$n$:** Índice (si es 2, se llama raíz cuadrada y no se escribe).
* **$a$:** Radicando (o cantidad subradical).
* **$b$:** Raíz.
* **$\sqrt{ \ }$**: Símbolo radical.

---

## Propiedades Fundamentales
Las raíces se pueden expresar como potencias de exponente fraccionario: $\sqrt[n]{a^m} = a^{\frac{m}{n}}$. Debido a esto, comparten propiedades similares:

| Propiedad | Fórmula |
| :--- | :--- |
| **Raíz de un producto** | $\sqrt[n]{a \cdot b} = \sqrt[n]{a} \cdot \sqrt[n]{b}$ |
| **Raíz de un cociente** | $\sqrt[n]{\frac{a}{b}} = \frac{\sqrt[n]{a}}{\sqrt[n]{b}}$ |
| **Raíz de una raíz** | $\sqrt[m]{\sqrt[n]{a}} = \sqrt[m \cdot n]{a}$ |
| **Potencia de una raíz** | $(\sqrt[n]{a})^m = \sqrt[n]{a^m}$ |

## El Caso de las Raíces Pares y el Signo

Una de las características más importantes de la radicación es cómo se comporta según el índice sea par o impar.

### 1. Las dos respuestas de las raíces pares
Cuando el índice es **par** ($2, 4, 6...$), cualquier número positivo tiene **dos raíces reales**: una positiva y una negativa.
* Ejemplo: $\sqrt{9} = \pm 3$
* **¿Por qué?** Porque tanto $(+3)^2 = 9$ como $(-3)^2 = 9$.

### 2. El problema de la raíz de $-1$ (Números Imaginarios)
En el conjunto de los **Números Reales ($\mathbb{R}$)**, no existe la raíz par de un número negativo.
* $\sqrt{-1} = \text{ No existe en } \mathbb{R}$
* **¿Por qué?** Porque no hay ningún número real que, al elevarlo al cuadrado (o a cualquier potencia par), resulte en un número negativo (recordemos que "menos por menos es más").

Para solucionar esto, las matemáticas crearon el conjunto de los **Números Imaginarios**, donde:
$$\sqrt{-1} = i$$

### 3. Raíces Impares
Las raíces de índice impar ($3, 5, 7...$) no tienen este problema. Conservan el signo del radicando y solo tienen una respuesta real.
* $\sqrt[3]{8} = 2$ (porque $2^3 = 8$)
* $\sqrt[3]{-8} = -2$ (porque $(-2)^3 = -8$)

## Simplificación y Racionalización
* **Simplificar:** Extraer factores del radical. $\sqrt{50} = \sqrt{25 \cdot 2} = 5\sqrt{2}$.
* **Racionalizar:** Proceso para eliminar raíces del denominador de una fracción, multiplicando por un factor adecuado.
