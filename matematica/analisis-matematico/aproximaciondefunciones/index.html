Aproximar Funciones

La aproximación por diferenciales utiliza la recta tangente en un punto conocido aaa para estimar el valor de una función en un punto cercano xxx. La fórmula fundamental es:

f(x)≈f(a)+f′(a)(x−a)f(x) \approx f(a) + f'(a)(x - a)f(x)≈f(a)+f′(a)(x−a)

Donde:

  • f(a)f(a)f(a): Valor exacto en el punto fácil de calcular.
  • f′(a)f'(a)f′(a): La pendiente (derivada) en ese punto.
  • (x−a)(x - a)(x−a): La distancia o error (Δx\Delta xΔx).

1. Pasos para la Aproximación

Sección titulada «1. Pasos para la Aproximación»

Para calcular un valor difícil (como 14\sqrt{14}14​):

  1. Definir la función: f(x)=xf(x) = \sqrt{x}f(x)=x​.
  2. Elegir un punto cercano “fácil” (aaa): Buscamos el cuadrado perfecto más cercano a 14, que es 161616 (o 9, pero 16 está más cerca).
  3. Calcular la derivada: f′(x)=12xf'(x) = \frac{1}{2\sqrt{x}}f′(x)=2x​1​.
  4. Evaluar en el punto fácil (a=16a=16a=16):
    • f(16)=16=4f(16) = \sqrt{16} = 4f(16)=16​=4.
    • f′(16)=1216=18=0.125f'(16) = \frac{1}{2\sqrt{16}} = \frac{1}{8} = 0.125f′(16)=216​1​=81​=0.125.
  5. Calcular la diferencia (Δx\Delta xΔx): x−a=14−16=−2x - a = 14 - 16 = -2x−a=14−16=−2.

2. Ejemplo: Calcular 14\sqrt{14}14​ con 3 cifras significativas

Sección titulada «2. Ejemplo: Calcular 14\sqrt{14}14​ con 3 cifras significativas»

Aplicamos la fórmula: f(14)≈f(16)+f′(16)⋅(14−16)f(14) \approx f(16) + f'(16) \cdot (14 - 16)f(14)≈f(16)+f′(16)⋅(14−16)

  1. Sustituir valores: f(14)≈4+(0.125)⋅(−2)f(14) \approx 4 + (0.125) \cdot (-2)f(14)≈4+(0.125)⋅(−2)
  2. Operar: f(14)≈4−0.25f(14) \approx 4 - 0.25f(14)≈4−0.25 f(14)≈3.75f(14) \approx 3.75f(14)≈3.75

Resultado: La aproximación es 3.75. (Nota: El valor real es ≈3.741\approx 3.741≈3.741. Nuestra aproximación tiene un error de apenas el 0.24%).

3. Cifras Significativas y Error

Sección titulada «3. Cifras Significativas y Error»

Al trabajar con aproximaciones, debemos respetar la precisión requerida:

  • Cifras significativas: En nuestro resultado 3.75, tenemos tres cifras significativas.
  • Error de aproximación: Cuanto más lejos esté xxx de aaa, mayor será el error, ya que la curva se separa de la recta tangente.

4. Aplicación en Ingeniería

Sección titulada «4. Aplicación en Ingeniería»

Este método es la base de:

  • Análisis de errores: Para saber cómo se propaga el error de una medición en un diseño.
  • Algoritmos de computación: Muchos procesadores usan métodos de aproximación (como el Método de Newton-Raphson) para resolver raíces y potencias rápidamente.
